# 달인 부킹 (Dalin Booking) - 제품 요구사항 문서 (PRD)

## 1. 프로젝트 개요

### 1.1 비전
"달인 부킹"은 1인 가구와 MZ 세대를 위한 생활 기술 서비스 플랫폼입니다. 배달의 민족이 음식 배달을 혁신했듯이, 우리는 개인 기술자(달인)와 서비스가 필요한 고객을 안전하고 빠르게 연결합니다.

### 1.2 핵심 가치 제안
- **신뢰와 안전**: 철저한 신원 검증, 범죄 경력 조회, 의무 보험 가입
- **즉시 해결**: AI 기반 추천 가격으로 견적 대기 없이 즉시 매칭
- **공정한 가격**: 데이터 기반 표준 가격으로 달인의 가치 보호

### 1.3 시장 차별화
| 항목 | 기존 경쟁사 (숨고) | 달인 부킹 |
|------|-------------------|-----------|
| 서비스 범위 | 광범위 (1,000개 이상 카테고리) | 생활 밀착형 소규모 수리/설치 집중 |
| 매칭 방식 | 역경매 (견적 대기 1-3일) | AI 추천 가격 + 즉시 매칭 |
| 안전성 | 후기/평점 중심 | 범죄 경력 조회 + 의무 보험 |
| 가격 구조 | 저가 경쟁 유도 | 표준 가격으로 달인 가치 보호 |

## 2. 타겟 사용자

### 2.1 고객 (서비스 이용자)
- **주요 타겟**: 20-40대 1인 가구, MZ 세대, 직장인
- **니즈**: 
  - 간단한 생활 수리/설치가 필요하지만 대기업 A/S는 과함
  - 집에 들일 기술자의 신원이 검증되길 원함
  - 빠른 해결을 원하며 합리적인 가격 지불 의사 있음

### 2.2 달인 (서비스 제공자)
- **주요 타겟**: 기술 보유 개인, 은퇴 숙련 기술자, 부업 희망자
- **니즈**:
  - 자유로운 시간에 추가 수입 창출
  - 자신의 기술 가치를 인정받고 싶음
  - 안정적인 일감 확보

## 3. 핵심 기능 (MVP)

### 3.1 고객 앱 기능

#### 3.1.1 서비스 요청
- 카테고리 선택 (수리/설치, 디지털 지원 등)
- 사진/영상 첨부 기능
- 희망 일정 및 위치 입력
- **부킹 옵션 선택**:
  - 일반 부킹 (표준 가격)
  - 우선 부킹 (+5-10% 할증)
  - 긴급 부킹 (+15-20% 할증)

#### 3.1.2 달인 매칭
- AI 추천 가격 즉시 표시
- 가까운 거리의 달인 자동 매칭
- 달인 프로필 확인 (인증 배지, 평점, 후기)
- 1:1 인앱 채팅

#### 3.1.3 결제 및 후기
- 사전 결제 (카드, 간편결제)
- 작업 완료 후 후기/평점 작성
- 서비스 내역 확인

### 3.2 달인 앱 기능

#### 3.2.1 일감 확보
- 실시간 작업 요청 알림 (푸시)
- 지리 기반 요청 목록
- AI 추천 가격 확인
- 원클릭 수락

#### 3.2.2 작업 관리
- 예약/작업 일정 캘린더
- 작업 진행 상태 변경 (수락 → 이동 중 → 작업 시작 → 완료)
- 고객과 1:1 채팅

#### 3.2.3 정산
- 작업별 수입 내역 확인
- 자동 정산 신청

### 3.3 AI 추천 가격 시스템

#### 3.3.1 가격 산출 요소
\`\`\`
[AI 추천 가격] = [기본 서비스 가격] + [지역 할증] + [시간대 할증] + [달인 등급] + [부킹 옵션 할증]
\`\`\`

#### 3.3.2 고려 요인
- 지역별 차이 (서울/지방, 도심/외곽)
- 시간대/요일 (심야, 주말, 공휴일)
- 달인 등급 (실버/골드/마스터 달인)
- 날씨/긴급성
- 부킹 옵션 (일반/우선/긴급)

## 4. 신뢰 및 안전 시스템

### 4.1 달인 검증 (필수)
- 실명 및 거주지 확인
- 휴대폰 본인 인증 + 계좌 인증
- **범죄 경력 조회 동의 의무화**
- 얼굴 사진 등록 의무
- 온라인 안전 교육 이수

### 4.2 서비스 중 안전
- **의무 보험 가입** (영업 배상 책임)
- 실시간 위치 정보 제공 (선택)
- 안심 문자 서비스 (가족 알림)
- 서비스 시작/종료 자동 알림

### 4.3 양방향 평가
- 고객 → 달인: 기술력, 친절도, 위생, 사생활 존중
- 달인 → 고객: 매너, 안전성
- 삼진 아웃 제도 (심각한 문제 발생 시 즉시 퇴출)

## 5. 수익 모델

### 5.1 기본 거래 수수료
- 서비스 금액의 15-25% 플랫폼 수취
- 초기 3개월 수수료 할인 (5-10%)
- 우수 달인 차등 수수료 적용

### 5.2 추가 수익원
- 프리미엄 노출 (달인 광고)
- 달인 멤버십 (교육, 보험 할인)
- 장비/부품 제휴 수수료

## 6. 서비스 카테고리 (초기)

- 문고리 수리

- 스마트 기기 설치/연동

### 6.3 취미/DIY 코칭
- 목공 기본 교육

- 특정 장비 사용법 교육
